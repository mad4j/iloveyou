# iloveyou
Javascript per San Valentino

## Caratteristiche

- Animazione interattiva della curva del cuore
- Progressive Web App (PWA) installabile
- **Pull-to-refresh**: scorri verso il basso dalla parte superiore dello schermo per aggiornare la pagina

## Pull-to-Refresh

L'applicazione supporta il refresh della pagina tramite swipe verso il basso:
1. Inizia il gesto dalla parte superiore dello schermo (entro 50px dall'alto)
2. Trascina verso il basso per almeno 100px
3. Rilascia per ricaricare la pagina

Questa funzionalità è particolarmente utile sui dispositivi mobili.
